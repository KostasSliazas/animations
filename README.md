## Installation

   
1. **Clone the repository:**
   ```bash
   git clone https://github.com/KostasSliazas/animations.git
   ```
2. **Open the project folder:**
   ```bash
   cd animations
   ```
3. **Open `index.html` in your browser** to explore the animations. For example:
   ```bash
   firefox index.html
